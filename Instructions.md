###### **Create a new Collection.**



* There are three ways to create a new collection: on the main screen, tap the “Create” button in the upper-right corner to create a collection titled with today's date; “Create from open tab” creates a collection from the tabs currently open in your browser; and the “New collection name” input field allows you to create a collection with a custom name.



* Adding a page to a collection is very simple: just tap the + button on the right side of a collection panel. To select a page, simply check the box below the + symbol.



* You can change a collection’s position using the collection sorting options in the main menu (the three dots in the top-right corner), or by holding down on a collection and dragging it next to another to reorder them.





###### **Collection Lock**



* To use password-protected collection locking, you must access the “Set Password” feature from the main menu \[click the three dots in the upper-right corner and set a master password using the “Set Password” feature]. Once you've done that, simply right-click on a collection and select “Lock Collection.” The collection will be locked, and you can unlock it by entering the password. You cannot relock the collection directly; you must first remove the lock using the “Remove Lock” function.





###### **Notes**



* In the top-left corner of the collection, there is a small icon shaped like a Post-it note or sticky note; clicking it opens the note editor. The note editor lets you write notes just as you would on a sticky note; its functions are very straightforward, typical of a text editor, and you can also change the color of a note using the color palette on the editor’s toolbar.



###### **Saving and Exporting**



* Pinboard includes a backup import system from Edge Collections. I’ve included a script \[edge\_collections\_import\_lite.py] that will convert your old Microsoft Edge Collections into a format compatible with Pinboard. It’s important to install Python on your computer; otherwise, the script won’t work. The script will save a .json file in the same directory from which it was run; this file must be imported using the “Import Backup” function in the main menu. Using the “Export Backup” function, you can save the entire collection tree for safekeeping.



* The main menu’s functions vary depending on whether you are on the Collections screen or the Pages or Items screen (within a collection). The main menu on the Pages screen includes functions for managing items within collections; you can export items in various formats, move them between collections, or add them directly to a new collection.





###### **Refresh preview**



* When a collection is imported, it’s possible that a page doesn’t exist or has had its content changed, causing the preview to fail to load properly. In this case, simply click the circular arrow icon labeled “refresh missing preview” in the upper-right corner of a collection. This feature will open three pages at a time in three different tabs without displaying them to you, searching only for the elements needed to create a valid preview. If the page no longer exists, the preview will display a “Not found” box. At the end of the scan, all “Not Found” pages will be preselected, and the export of a text file will begin. This file will contain the information from these “Not Found” pages before they are overwritten by the new regeneration. This feature is designed to give you the opportunity to try finding elsewhere what no longer exists on the page—perhaps a video, a tutorial, or a software program—and to help you remember the name of what you were looking for.



###### **Menù**



* Pinboard features several menus; two of them are general menus and can be accessed either on the collection screen in the top-right corner by clicking the three vertical dots, or on the pages screen (within a collection) in the same way as the first. The other menus are context-sensitive; you can access them by right-clicking either on a collection or on an item within the collection—such as a page or note. There is another context menu that you can open by clicking in the area outside the items in a collection. Each menu has functions specific to that particular area, although some are shared—for example, the sorting system for items in the two general menus.



###### **Language**



* Pinboard has been translated into eight different languages in an effort to accommodate everyone. The primary language is detected automatically; if that doesn't happen, you can always change it manually from the main menu on the first screen.
